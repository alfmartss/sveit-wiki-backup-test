Basic help on editing this wiki

  • You must login to edit contents
  • Creating Pages: 3 types of pages, “Visual Editor” are the easy ones.
  • Folders: for organization and permissions.
  • Internal links: select text and use the button “Insert link into page” then select the page you want to link.
  • External links: select text and use the button “Link”.
  • Internal Images: use “Insert Assets” button, then upload new images or select existing ones. Take care of the size of the images as they are stored on the wiki-server.
  • External Images: the “Visual Editor” pages dont support external images but they can be tweaked.
