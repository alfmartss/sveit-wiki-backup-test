Why

  • The visual editor doesnt support external images
  • Internal images are stored in the wiki server and i hasnt infinite space
  • Considerations: if the external image can be deleted, it wont show here. Consider copying it to another place. 

 

How to

  1. Insert an internal image using an existing one. 
  2. Convert the page to ”Markdown"
  3. Edit the page and search in the code for something like this : ![image caption text](/imagexXXX.jpg).  
  4. Delete the internal path betweend ( ) and copy the external link, something like https://media.discordapp.net/attachments/827919413809643620/833345859659366430/unknown.png?width=832&height=493
  5. The preview of the page should show the new image used. 
  6. If the image loads slowly consider using a smaller one 
  7. Save the page
  8. Convert the page to “Visual Editor”
  9. From now on you can edit your content, even Cut-Paste the image.
