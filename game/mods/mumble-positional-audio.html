Intro

This mod makes it really interesting. You hear people in the woods screaming or a village full with talking people?

ALWAYS START MUMBLE AND THEN THE GAME

Installation

This is how you install this

Mumble

  1. Download the Mumble app: https://www.mumble.info/downloads/
  2. Install mumble and add the Server to your list (Server info at the bottom)
  3. When you joined the server head to:
    3.1 Configure - Settings - Audio Output - "Positional Audio" (Checked)
    3.2 Position Audio - min Distance 1m - Max Distance 50m - Minimum Volume 0%
    3.3 Configure - Settings - Plugins - "Link to Game and Transmit Position" (Checked)
  4. Join the "Valheim" room and talk to the others

Now all that will happen is that you will hear the people from there position and with that you will be able to have even more atmosphere.

Travel with people, meet new people and enjoy everything around you. Listen to music in the Discord room while travelling around or screaming while running away from the enemies.

Optional settings

No sounds/messages when something happens

Go into "Configure - Settings - Messages" and uncheck the "Text-To-Speech" boxes. Then the program does not talk to you while playing.

Hotkey for Mute

You have your voice always activated so you do not have to push even more buttons? Do this to add a hotkey for mute.
Go into "Configure - Settings - Shortcuts" and add a "Mute Self" hotkey
